# 🖐️ AI Hand Gesture Smart Home Control
A touchless human-computer interaction system designed for **Smart Home Automation** using Computer Vision.

### **Overview**
This project uses **OpenCV** and **MediaPipe** to detect hand landmarks and recognize gestures. It can be integrated with IoT devices to control lights, fans, or volume using simple hand movements.

### **Technical Stack**
* **Language:** Python
* **Computer Vision:** OpenCV
* **Tracking Library:** cvzone (Powered by MediaPipe)

### **Key Features**
* **Real-time Tracking:** Detects 21 hand points for high accuracy.
* **Gesture Logic:** Map specific finger patterns to different commands.
* **Zero Hardware Cost:** Works using a standard laptop webcam.

### **How to Run**
1. Install requirements: `pip install -r requirements.txt`
2. Run the script: `python main.py`
